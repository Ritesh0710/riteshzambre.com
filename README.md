Since this is a static website, you can host it for **FREE** using GitHub Pages.

### Step 1: Create a GitHub Repository
1. Go to [GitHub.com](https://github.com) and sign in.
2. Click the **+** icon in the top right and select **New repository**.
3. Name it `portfolio` (or `riteshzambre.com` if you want to use that domain later).
4. Make sure it is **Public**.
5. Click **Create repository**.

### Step 2: Upload Files
1. Open the folder `C:\Users\rites\.gemini\antigravity\scratch\RiteshPortfolio` on your computer.
2. In the GitHub repository page, click **uploading an existing file**.
3. Drag and drop **ALL** files and folders (`index.html`, `css/`, `js/`, `assets/`, `README.md`) into the browser window.
4. Wait for them to upload, then click **Commit changes**.

### Step 3: Enable GitHub Pages
1. In your repository, go to **Settings** (top tabs).
2. on the left sidebar, click **Pages**.
3. Under **Build and deployment** > **Source**, select **Deploy from a branch**.
4. Under **Branch**, select `main` (or `master`) and folder `/ (root)`.
5. Click **Save**.

### Step 4: Live Site
- Refresh the page after a minute.
- You will see a link at the top (e.g., `https://yourusername.github.io/portfolio/`).
- Click it to see your live website!
